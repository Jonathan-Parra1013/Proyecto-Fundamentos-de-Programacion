Esta función tiene como objetivo solicitar al usuario el nombre del equipo cuyos jugadores desea consultar.

9. Conclusiones
* Las lecciones aprendidas durante el desarrollo del proyecto fueron que al programar una pagina web no solo se requiere escribir el codigo, sino tambien planear bien como se va a organizar la informacion y como se va a mostrar al usuario. Entendimos la importancia de trabajar en orden, organizar el codigo y dividir las tareas para avanzar de manera mas eficiente. Tambien aprendimos a manejar mejor la logica de programacion y las estructuras de datos, ademas de esto, comprendimos que probar constantemente el proyecto es clave para detectar errores a tiempo y mejorar el resultado final.
* Las dificultades que nosotros encontramos fueron que habia que organizar demasiada informacion, es decir, organizar todo los datos de cada jugador y de cada equipo, esto hizo que en algunas partes del proceso de creacion del proyecto nos estresaramos y decidieramos no seguir, pero nos demostramos a nosotros mismos que con paciencia y trabajo en equipo se pueden resolver las cosas. Otra dificultad fue que al principio los datos no se mostraban correctamente y aparecian en desorden o algunos datos no eran del jugador selecto. Tuvimos algunos problemas con el diseño visual de la pagina, espeficidamente para que fuera facil de entedner y se viera bien, esto lo solucionamos revisando la estrucutra del codigo y aplicando estilos mas claros y organizados. La mayor dificultad nuestra fue la coordinacion de las tareas, es decir, que todas los integrantes del grupo hicieramos algo y que no solo 1 o 2 hicieran todo y lo solucionamos mediante la comunicacion verbal y haciendo una lista de lo que tocaba hacer y ver si todos ya teniamos una tarea asignada
* Las posibles mejoras o ideas futuras de este proyecto son que como grupo nos gustaria añadir mas funciones a la pagina, como estadisticas en tiempo real, actualizaciones automaticas de los partidos y agregar graficos de las estadisticas para realizar comparaciones entre jugadores y equipos, esto para que la informacion se "ejecute" de manera mas visual y dinamica. Otra mejora seria crear una base de datos en linea que se actualice automaticamente, de manera que los datos no tengan que modificarse manualmente. Ademas, podriamos optimizar el diseño para hacerlo aun mas moderno y atractivo, es decir, colocar el logo de la premier league y/o crear nuestro propio logo el cual nos ayudaria a que las personas nos identifiquen de manera rapido y sencilla
